Detailed Description

Represents the constraints for placing UI controls arbitrarily within an XYLayout.

This class defines the position constraint, which is necessary to position UI elements in specific locations within an XYLayout. Each Control within the Layout can have a unique constraint, specifying its position inside its Container.
