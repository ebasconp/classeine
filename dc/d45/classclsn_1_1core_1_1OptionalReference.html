Detailed Description

template<typename T>
class clsn::core::OptionalReference< T >

A class that encapsulates an optional reference to an object of type T.

This class uses std::optional and std::reference_wrapper to manage optional references to objects.

Template Parameters
TThe type of the object the reference will Point to.
