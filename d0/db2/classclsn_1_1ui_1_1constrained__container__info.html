Detailed Description

template<typename Constraint>
class clsn::ui::constrained_container_info< Constraint >

Holds a control and its associated constraint.

This class is a wrapper that contains a pointer to a control and an associated layout constraint. It provides access to the control and constraint, allowing for constrained layout management in a UI.

Template Parameters
ConstraintThe type of the layout constraint associated with the control.
