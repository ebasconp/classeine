Converts the color to grayscale if the specified condition is true.

This method helps simplify color decision logic by embedding the condition check and the conversion in a single method. If the condition is true, the color is converted to its grayscale equivalent.

Parameters
conditionBoolean condition to trigger the grayscale conversion.
Returns
A new color object in grayscale if condition is true, otherwise the original color.
