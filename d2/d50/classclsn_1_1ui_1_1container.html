Detailed Description

template<typename ElementType>
class clsn::ui::container< ElementType >

Represents a generic container for UI elements.

This class serves as a container that can hold UI elements along with their layout constraints. The key difference between an Element and a Control is that an Element is the entity stored in the container, likely representing a control with additional information, such as layout constraints.

Template Parameters
ElementTypeThe type of elements stored in the container.
