Detailed Description

template<typename T, bool HasValueChangedEventListener>
class clsn::core::property< T, HasValueChangedEventListener >

Template class representing a property with an optional state change listener.

The property class template manages a property's value and provides mechanisms to listen for changes to the property's value.

Template Parameters
TThe type of the property value.
HasValueChangedEventListenerIndicates if the property has value change listeners.
